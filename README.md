Project Overview 

This repository contains my Digital Literacy Project Portfolio, created as part of the CSE0001 course. The project is based on the role of a Student Digital Ambassador, where the objective is to promote responsible and effective use of digital tools among peers.  

The project focuses on developing essential digital skills such as: 

Understanding digital literacy concepts  

Building a professional online presence  

Practicing collaboration and coding tools  

Learning professional communication  

Awareness of cybercrime and online safety 

 

 

 

 

 

 

 

Objectives 

Develop awareness about digital responsibility  

Create and manage a professional digital identity  

Improve communication and collaboration skills  

Understand online risks and cyber safety practices  

Apply digital tools effectively in academic life 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Task Summary 

📊 Task 1: Digital Literacy Infographic

Focused on simplifying digital literacy concepts 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Task 2: Digital Portfolio 

Created/updated profiles on platforms such as GitHub and LinkedIn  

Added academic details and basic professional information

💻 Task 3: Coding & Collaboration Platforms 

Completed a beginner coding challenge on platforms like HackerRank  

Created a Google Form quiz for digital literacy awareness  

Learned how collaboration tools support academic work 

 

Task 4: Email Etiquette 

Drafted two professional emails:  

Request for assignment extension  

Internship application email  

Created a social media Do’s & Don’ts checklist  

Understood importance of professional communication  

 

🔐 Task 5: Cybercrime Awareness 

Developed a case study on cybercrime (e.g., phishing)  

Created a prevention checklist for online safety  

Learned how to identify and avoid digital threats 

 

 

Key Learnings 

Digital literacy is not just about using tools but understanding their impact  

Online identity must be managed responsibly  

Professional communication is essential in academic and career growth  

Cyber awareness is critical in today’s digital world
